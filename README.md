## вҡҷпёҸ Installation

### Clone the Repository

```bash
git clone https://github.com/R-Samaikya/MediCare.git
```

### Navigate to Project

```bash
cd MediCare
```

### Install Frontend Dependencies

```bash
cd client
npm install
npm run dev
```

### Install Backend Dependencies

```bash
cd ../server
npm install
npm start
```
